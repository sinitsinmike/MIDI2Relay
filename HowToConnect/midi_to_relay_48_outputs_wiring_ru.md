# MIDI to Relay / Opto — подключение конкретных модулей 74HC595

## Важно

В этом проекте используются **2 розовых модуля 74HC595**.

На каждом таком модуле установлено:

- 3 микросхемы 74HC595
- 24 логических выхода: `Q0–Q23`

Итого:

```text
2 модуля × 24 выхода = 48 выходов
```

В прошивке это соответствует:

```cpp
#define NUM_SHIFT_REGISTERS 6
#define FIRST_NOTE 36
```

Значит рабочий диапазон MIDI-нот:

```text
36–83
```

---

# 1. Подписи на твоем модуле 74HC595

На твоем модуле подписаны такие входные/служебные пины:

```text
LDEN
GND
VCC
DSI
LDSTR
LDSCK
```

На другом конце модуля есть похожий разъем, но вместо `DSI` там `QD`:

```text
LDEN
GND
VCC
QD
LDSTR
LDSCK
```

---

# 2. Что означает каждая надпись

| Надпись на модуле | Что это значит |
|---|---|
| `VCC` | +5V питание логики |
| `GND` | земля |
| `DSI` | вход данных в модуль |
| `QD` | выход данных на следующий модуль |
| `LDSCK` | тактовый сигнал / CLOCK |
| `LDSTR` | защелка / LATCH |
| `LDEN` | разрешение выходов / ENABLE |

---

# 3. Arduino Nano → первый модуль 74HC595

Подключить строго так:

| Arduino Nano | Первый модуль 74HC595 |
|---|---|
| `D11` | `DSI` |
| `D13` | `LDSCK` |
| `D10` | `LDSTR` |
| `GND` | `LDEN` |
| `5V` | `VCC` |
| `GND` | `GND` |

## Простыми словами

```text
D11 Arduino  → DSI модуля
D13 Arduino  → LDSCK модуля
D10 Arduino  → LDSTR модуля
GND Arduino  → LDEN модуля
5V Arduino   → VCC модуля
GND Arduino  → GND модуля
```

`LDEN` обязательно подключить к `GND`, чтобы выходы модуля были включены.

---

# 4. Первый модуль 74HC595 → второй модуль 74HC595

Второй модуль подключается к первому так:

| Первый модуль | Второй модуль |
|---|---|
| `QD` | `DSI` |
| `LDSCK` | `LDSCK` |
| `LDSTR` | `LDSTR` |
| `LDEN` | `LDEN` |
| `VCC` | `VCC` |
| `GND` | `GND` |

## Простыми словами

```text
QD первого модуля     → DSI второго модуля
LDSCK первого модуля  → LDSCK второго модуля
LDSTR первого модуля  → LDSTR второго модуля
LDEN первого модуля   → LDEN второго модуля
VCC первого модуля    → VCC второго модуля
GND первого модуля    → GND второго модуля
```

Так получается цепочка:

```text
Arduino Nano → модуль 74HC595 #1 → модуль 74HC595 #2
```

---

# 5. Выходы на нижней гребенке

На каждом розовом модуле уже выведены логические выходы:

```text
Q0–Q23
```

Это готовые выходы с микросхем 74HC595.

У двух модулей всего:

```text
Q0–Q23 на первом модуле
Q0–Q23 на втором модуле
```

В сумме:

```text
48 логических выходов
```

# 6. Подключение 74HC595 → ULN2803

Каждый ULN2803 имеет 8 входов:

```text
IN1
IN2
IN3
IN4
IN5
IN6
IN7
IN8
```

Один ULN2803 обслуживает 8 выходов.

Для 48 выходов нужно:

```text
6 модулей ULN2803
```

# 9. Подключение ULN2803 к реле

ULN2803 не подает плюс.

ULN2803 замыкает выход на землю.

## Один канал реле

```text
первый вывод катушки реле  → +12V или +24V
второй вывод катушки реле  → OUT1 ULN2803
```

Для следующего канала:

```text
первый вывод катушки реле  → +12V или +24V
второй вывод катушки реле  → OUT2 ULN2803
```

И так далее.

---

# 10. COM на ULN2803

Пин `COM` на каждом ULN2803 подключить к плюсу питания реле:

```text
COM ULN2803 → +12V или +24V
```

Это нужно для работы встроенных защитных диодов.

---

# 11. GND

Все земли должны быть соединены вместе:

```text
GND Arduino Nano
GND модулей 74HC595
GND модулей ULN2803
минус блока питания реле
```

Итог:

```text
GND Arduino = GND 74HC595 = GND ULN2803 = минус питания реле
```

Без общего GND система нормально работать не будет.

---

# 12. Питание

## Нельзя

```text
Нельзя питать реле от Arduino Nano.
```

## Правильно

| Устройство | Питание |
|---|---|
| Arduino Nano | 5V |
| 74HC595 модули | 5V |
| Реле | отдельный блок питания 12V или 24V |

# 13. Соответствие MIDI-нот и выходов

В прошивке:

```cpp
#define FIRST_NOTE 36
#define NUM_SHIFT_REGISTERS 6
```

Значит:

```text
6 × 8 = 48 выходов
```

Диапазон нот:

```text
36–83
```

| MIDI Note | Выход |
|---|---|
| 36 | Output 1 |
| 37 | Output 2 |
| 38 | Output 3 |
| 39 | Output 4 |
| ... | ... |
| 83 | Output 48 |

---

# 14. Комментарий для прошивки

```cpp
/*
  По умолчанию код читает все MIDI-каналы.

  Используется 2 розовых модуля 74HC595.
  На каждом модуле установлено 3 микросхемы 74HC595.
  Всего физических shift registers: 6.
  Всего выходов: 6 × 8 = 48.

  При FIRST_NOTE = 36 используются MIDI ноты 36–83.
*/
```

---

# 15. Настройки прошивки

```cpp
// Количество физических 74HC595
// 1 розовый модуль = 3 регистра = 24 выхода
// 2 розовых модуля = 6 регистров = 48 выходов
#define NUM_SHIFT_REGISTERS 6

// Всего выходов
#define TOTAL_OUTPUTS (NUM_SHIFT_REGISTERS * 8)

// Первая MIDI нота
#define FIRST_NOTE 36
```

---

# 16. Краткая схема соединений

```text
MIDI IN
  ↓
оптопара 6N137 / H11L1 / 6N138 / 6N139
  ↓
Arduino Nano
  ↓
DSI / LDSCK / LDSTR
  ↓
74HC595 модуль #1
  ↓ QD → DSI
74HC595 модуль #2
  ↓
Q0–Q23 + Q0–Q23
  ↓
ULN2803 × 6
  ↓
48 реле / оптрона / внешних нагрузок
```

---

# 17. Минимальная проверка перед подключением реле

1. Подать питание 5V на Arduino и 74HC595.
2. Проверить общий GND.
3. Загрузить прошивку.
4. Подать MIDI-ноты 36–83.
5. Проверить выходы Q0–Q23 мультиметром или светодиодом через резистор.
6. Только после этого подключать ULN2803 и реле.

# 18. Важное предупреждение

Не подключать 220V напрямую к макетной сборке.

Для сетевого напряжения нужны:

- нормальная PCB
- зазоры по безопасности
- предохранитель
- корпус
- клеммы
- изоляция
- понимание работы с опасным напряжением
